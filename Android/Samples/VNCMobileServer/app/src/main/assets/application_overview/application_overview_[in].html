Ringkasan Aplikasi

T-Link adalah aplikasi berbagi layar smartphone yang dirancang untuk Display Audio.
Dengan kemampuan kontrol sentuh 2 arah yang nyaman.
T-Link memaksimalkan pengalaman berkendara Anda.

[Bagaimana menghubungkan Display Audio]

Berbagi suara: melalui koneksi Bluetooth
Berbagi layar: dengan koneksi kabel USB

[Keterangan]

Dapat membagikan aplikasi smartphone apa pun ke Display Audio.
Beberapa operasi dari sisi Display Audio dibatasi saat berkendara tergantung pada aplikasi.
Beberapa fungsi mungkin tidak berfungsi dengan operasi pada Display Audio tergantung pada perangkat yang terhubung.

[Perangkat yang kompatibel]

Android OS ver 5.0 atau lebih tinggi.
Kernel versi 3.5 atau lebih tinggi.

[Lainnya]

Aplikasi ini menggunakan izin berikut.
- Layanan Aksesibilitas
- Tampilan di atas aplikasi lain
