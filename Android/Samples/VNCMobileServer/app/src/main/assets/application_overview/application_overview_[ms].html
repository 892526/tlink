Gambaran Keseluruhan Aplikasi

T-Link adalah aplikasi perkongsian skrin telefon pintar direka untuk Audio Paparan.
Dengan keupayaan kawalan sentuhan 2-cara yang mudah.
T-Link memaksimumkan pengalaman dalam kereta anda.

[Bagaimana untuk menyambung dengan Audio Paparan]

Perkongsian suara: melalui sambungan Bluetooth
Perkongsian skrin: dengan sambungan kabel USB

[ Kenyataan ]

Boleh berkongsi aplikasi telefon pintar untuk Paparan Audio.
Sesetengah operasi dari sisi Paparan Audio adalah terhad semasa memandu kereta bergantung kepada aplikasi.
Sesetengah fungsi munkin tidak berfungsi melalui operasi pada Audio Paparan bergantung kepada sambungan peranti.

[Pernati yang serasi]

Android OS ver 5.0 atau keatas.
Kernel versi 3.5 atau keatas.

[Lain-lain]

Aplikasi ini menggunakan kebenaran seperti berikut.
- Perkhidmantan Akses
- Paparan keatas aplikasi lain
