ภาพรวมของแอปพลิเคชัน

T-Link เป็นแอปพลิเคชันสำหรับการแชร์หน้าจอสมาร์ทโฟน
ถูกออกแบบมาสำหรับเครื่องเสียง ด้วยความสามารถที่สะดวกในการควบคุม 2 ทาง
T-Link จะทำให้คุณได้สัมผัสประสบการณ์ในรถยนต์อย่างสูงสุด

[การเชื่อมต่อกับเครื่องเสียง]

การแชร์เสียง: ด้วยการเชื่อมต่อ Bluetooth
การแชร์หน้าจอ: ด้วยการเชื่อมต่อสาย USB

[ข้อสังเกต]

สามารถแชร์แอปพลิเคชันสมาร์ทโฟนใดๆก็ตามไปยังเครื่องเสียงได้
การทำงานบางอย่างจากเครื่องเสียงจะถูกจำกัดขณะขับขี่รถยนต์ ขึ้นอยู่กับแอปพลิเคชัน
ฟังก์ชันบางอย่างอาจไม่สามารถทำงานบนเครื่องเสียงได้ ขึ้นอยู่กับการเชื่อมต่อของอุปกรณ์

[อุปกรณ์ที่รองรับ]

Android OS เวอร์ชัน 5.0 หรือสูงกว่า
Kernel เวอร์ชัน 3.5 หรือสูงกว่า

[อื่นๆ]

แอปพลิเคชันนี้ใช้การอนุญาตดังต่อไปนี้
- การเข้าถึงของบริการ
- การแสดงผลบนแอปพลิเคชันอื่นๆ
