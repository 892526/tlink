Remote View Service for Android
===============================

This library project contains a Remote View Service suitable for use
on Android devices. It is a form of Remote Control Service which
provides screen capture but no event injection functionality. It can
be used on any device running Android 5.0 (Lollipop) and above.

To make use of this library project it must be included in the same
application as the server SDK.
